# Ractysh OTC Exchange

Next.js App Router starter for the Ractysh OTC exchange site.

## Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion
- Lucide icons
- Aceternity/shadcn-ready structure with `components.json`

## Scripts

```bash
npm run dev
npm run lint
npm run typecheck
npm run build
npm run start
```

The local development server can be started on a custom port with:

```bash
npm run dev -- --hostname 0.0.0.0 --port 3007
```
